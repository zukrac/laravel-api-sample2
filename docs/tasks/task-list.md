# Подготовка

  

1. Создайте новый публичный репозиторий на Gitlab.com, его нужно назвать “test29”. Пришлите на него ссылку по итогу тестового.
    
2. Установите последнюю версию фреймворка Laravel (12.x).
    

# Модели данных

  

1. Марка автомобиля
	1. - Название
2. Модель автомобиля
	1. - Название
	2. - Марка
3. Автомобиль
	1. - Марка
	2.  Модель
	3.  Год выпуска (опциональный атрибут)
	4.  Пробег (опциональный атрибут)
	5. - Цвет (опциональный атрибут)
    

# API

  

1. Список марок автомобилей;
    
2. Список моделей автомобилей;
    
3. Список + CRUD автомобилей.
